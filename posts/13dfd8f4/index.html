简介 GitHub Actions

GitHub Actions 是 GitHub 官方推出的持续集成/部署模块服务（CI/CD），只需一个配置文件即可自动开启服务，而且自带云环境运行。

基本概念

  1. workflow：持续集成一次运行的过程。
  2. job：一个 workflow 由一个或多个 job 构成。
  3. step：每个 job 由多个 step 构成，一步步完成。
  4. action：每个 step 可以依次执行一个或多个action。
